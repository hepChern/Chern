Chern documentation
=================================
在高能物理实验的数据分析工作中，我们常常需要面对的是对于数据采用不同的方法进行处理。
然而，随着实验的进展，我们需要不断地进行参数和模型的更新等工作。这时，不断的更新会
导致我们的程序与数据越发凌乱。我们在此希望提供的是一个帮助大家对分析进行项目管理的
软件框架，使大家的数据分析工作更加条理清楚。
